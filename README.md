# gtm-template-jtracking-logger-tag
Logs requests in GTM server

## Purpose
This template is designed for debugging server-side GTM requests by printing structured logs to the console.

## Data Handling
This template does not transmit data externally, does not persist data, and does not access cookies or storage.

## Privacy
No personal data is collected, stored, or shared. All logs are printed locally to the server-side GTM console only.
